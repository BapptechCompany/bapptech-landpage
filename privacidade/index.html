Privacidade

Política de Privacidade

Última atualização: 18/02/2026

1. Dados coletados

Coletamos os dados que você informa voluntariamente em formulários, como nome, e-mail, empresa, telefone e descrição do cenário.

2. Finalidade

Utilizamos os dados para responder solicitações, organizar diagnóstico técnico e manter comunicação sobre os serviços solicitados.

3. Base de tratamento

Tratamos os dados com base no consentimento informado no formulário e no legítimo interesse para atendimento das solicitações.

4. Compartilhamento

Não comercializamos dados. O compartilhamento ocorre apenas com fornecedores necessários para operação do atendimento.

5. Segurança e retenção

Aplicamos medidas proporcionais para proteger os dados e mantemos informações apenas pelo período necessário ao atendimento e obrigações legais.

6. Direitos do titular

Você pode solicitar acesso, correção, exclusão ou revogação de consentimento por meio do contato oficial.

7. Contato para privacidade

Para dúvidas sobre esta política: contato@bapptech.com.br
